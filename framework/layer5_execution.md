# Layer 5: Execution Strategy (执行策略)

## English

### Important Discovery
Serenity is a **day trader** (not a long-term investor)!

### Serenity's Words

> "I'm a day trader by profession. I use supply chain analysis to find the right stocks, then trade around my core positions."

### Core-Satellite Strategy

```
70% of capital: Long-term hold (Core + Swing positions)
30% of capital: Day trading (around core positions)
```

### Day Trading Logic

1. **Use supply chain information advantage** (front-run market reaction)
   - Example: Before Marvell earnings, predict results based on supply chain intelligence

2. **Trade around core positions** (reduce risk)
   - Buy dips in core positions
   - Sell rallies in core positions
   - Always maintain core position size

3. **Risk control**
   - Daily loss limit: **-2% of total capital**
   - Daily profit target: **+1% of total capital**
   - "I take profits fast. Day trading is not about home runs, it's about singles."

### Execution Checklist

#### Before Entry
- [ ] Thesis written down (1-2 sentences)
- [ ] Conviction level assigned (Core/Swing/Lottery)
- [ ] Position size calculated
- [ ] Stop loss level set
- [ ] Thesis validation checkpoints defined

#### After Entry
- [ ] Set alerts for thesis validation checkpoints
- [ ] Track news and earnings related to thesis
- [ ] Review position weekly
- [ ] Document any thesis changes

#### Exit Strategy
- [ ] **Profit-taking**: Trim at +100%, +200%
- [ ] **Stop-loss**: Exit at -20% (Swing) or -30% (Core)
- [ ] **Thesis broken**: Exit immediately (no questions asked)

### Serenity's Trading Rules

1. **Never average down** (only average up for Core positions)
2. **Take profits fast** (day trading is about singles, not home runs)
3. **Cut losses faster** (stop-thesis, not stop-loss)
4. **Stay disciplined** (follow the framework, don't chase momentum)

## 中文

### 重要发现
Serenity 是**日内交易员**（不是长期投资者）！

### Serenity 的原话

> "我是职业日内交易员。我用供应链分析找到正确的股票，然后围绕我的核心仓位交易。"

### 核心-卫星策略

```
70% 资金：长期持有（核心 + 摆动仓位）
30% 资金：日内交易（围绕核心仓位）
```

### 日内交易逻辑

1. **利用供应链信息优势**（提前市场反应）
   - 示例：在 Marvel 发布财报前，根据供应链信息预判业绩

2. **围绕核心仓位交易**（降低风险）
   - 在核心仓位的低点买入
   - 在核心仓位的高点卖出
   - 始终保持核心仓位大小

3. **风险控制**
   - 单日亏损上限：**-2% 总资金**
   - 单日盈利目标：**+1% 总资金**
   - "我快速获利。日内交易不是关于本垒打，而是关于一垒打。"

### 执行检查清单

#### 入场前
- [ ] Thesis 写下来（1-2 句话）
- [ ] 分配确信度水平（核心/摆动/彩票）
- [ ] 计算仓位大小
- [ ] 设置止损水平
- [ ] 定义 thesis 验证检查点

#### 入场后
- [ ] 为 thesis 验证检查点设置提醒
- [ ] 跟踪与 thesis 相关的新闻和财报
- [ ] 每周审查仓位
- [ ] 记录任何 thesis 变化

#### 退出策略
- [ ] **获利了结**：在 +100%、+200% 时减仓
- [ ] **止损**：在 -20%（摆动）或 -30%（核心）时退出
- [ ] **Thesis 破坏**：立即退出（不问问题）

### Serenity 的交易规则

1. **永远不要摊平亏损**（仅对核心仓位追涨）
2. **快速获利**（日内交易是关于一垒打，不是本垒打）
3. **更快止损**（stop-thesis，不是 stop-loss）
4. **保持纪律**（遵循框架，不要追逐动量）
